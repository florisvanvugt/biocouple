

# biocouple : determine coupling in biosignals


# Installation

Install latest development version from Github:

```
python -m pip install --upgrade "biobabel @ git+https://github.com/florisvanvugt/biobabel"
```


Install locally:

```
pip install .
```

and editable:

```
pip install -e .
```




# Usage

## Python
Simple usage example in Python:

```
import biocouple
```
